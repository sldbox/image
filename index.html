개복디 넥서스 사용 가이드

화면 구성

  • 상단 헤더에 사이트 아이콘 · 제목 · 제작자 정보가 표시됩니다. 오른쪽 [초기화]로 전체 리셋, [사용방법]으로 이 가이드를 열 수 있습니다.
  • 하단은 3개 패널로 구성됩니다 — 유닛 도감(왼쪽) · 계보 & 조합 트리(가운데) · 코스트 & 정수 매트릭스(오른쪽).
  • 모바일 세로: 매트릭스 → 도감 → 계보 순으로 세로 배치. 가로 회전 시 PC 3열 레이아웃으로 자동 전환됩니다.

유닛 도감 — 종족 탭

  • 종족 탭은 3×2 그리드(테바·테메·토바 / 토메·저그중립·혼종)로 배치됩니다. 각 탭 아이콘은 종족별로 구분됩니다.
  • 탭에 선택된 유닛이 있으면 초록 테두리 + 선택 수량 뱃지로 강조 표시됩니다.
  • 목록은 레전드 · 히든 · 슈퍼히든 등급만 표시되며, 총 매직코스트 높은 순으로 자동 정렬됩니다.

유닛 도감 — 카드 선택 & 수량

  • 각 카드는 위: 등급태그 + 유닛명 / 아래: 조합조건(재료명·조건·수량 가로 나열) 2줄 구조입니다.
  • 카드를 클릭하면 선택되어 도감 상단 [선택됨] 영역으로 이동합니다. 다시 클릭하면 해제되어 목록으로 복귀합니다.
  • 카드 오른쪽 [−] 숫자 [+]로 제작 수량을 1~8 사이에서 조절합니다. 여러 유닛 선택 시 코스트가 자동 합산됩니다.
  • 오늘 정수를 이미 확보한 유닛은 유닛명 옆에 [정수확보] 배지가 표시됩니다.

계보 & 조합 트리

  • 유닛을 선택하면 계보 트리가 자동 표시됩니다. 기본 상태는 1단계 직속 재료만 펼쳐지며, 비용은 최하단에 표시됩니다.
  • 각 재료 노드는 유닛명 · 조건(노란색) · 수량(초록색) · ▶ 가 하나의 버튼으로 통합됩니다. 클릭하면 하위 재료가 1단계씩 펼쳐집니다.
  • 카드 유닛명 제목을 클릭하면 전체 접기 ↔ 1단계 펼치기를 토글합니다.
  • [모두 펼치기] 버튼으로 전체 계보를 한번에 펼칩니다. 다시 클릭하면 1단계 상태로 복귀합니다.
  • 단일 유닛 선택 시 해당 유닛을 재료로 쓰는 상위 조합 유닛 목록이 트리 위에 추가 표시됩니다.

코스트 & 정수 통합 매트릭스 (5×5)

  • 1행: 총 매직코스트 · 총 정수합산 · 코랄(테바·테메) · 아이어(토바·토메) · 제루스(저그중립) 정수를 한눈에 표시합니다.
  • 2~5행 (20칸): 선택 유닛 제작에 필요한 기초 매직 유닛 수를 역산해 표시합니다. 값이 있는 칸은 초록색으로 활성화됩니다.
  • 혼종 유닛은 코랄·아이어·제루스 세 종족 정수에 모두 중복 집계됩니다.
